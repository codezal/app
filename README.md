# Codezal

Türkiye'den çıkan ışık — multi-LLM masaüstü sohbet uygulaması.

Tauri 2 + React + TypeScript + Tailwind. OpenAI, Anthropic, Google, DeepSeek desteği.

## Geliştirme

```bash
npm install
npm run tauri dev
```

## Build

```bash
npm run tauri build
```

## Stack

- **Tauri 2** — Rust shell, küçük binary
- **React 19 + TypeScript + Vite**
- **Tailwind CSS** — utility-first styling
- **Vercel AI SDK** — multi-provider abstraction
- **lucide-react** — icon set

## API anahtarları

İlk açılışta Ayarlar panelinden gir. localStorage'da saklanır.
