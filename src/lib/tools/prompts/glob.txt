Fast file pattern matching across the workspace that works with any codebase size, gitignore-aware (ripgrep --files when available, find fallback).

Usage:
- Supports glob patterns like "**/*.ts" or "src/**/*.tsx".
- Returns matching workspace-relative paths, sorted by modification time (most recently changed first), capped at 100 results.
- Use this tool when you need to find files by name pattern.
- You have the capability to call multiple tools in a single response. It is always better to speculatively run several searches as a batch when they are potentially useful.
